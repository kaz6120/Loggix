Basics

What is Loggix?

Loggix is a log managemant system runs on PHP5+.

  • Entry Management
    • You can post, edit, delete, search and add tags.
    • Archive searching with calendar.
    • Upload attachments.
      No limitations by file types. you can updoad any kinds of files.
    • User editable comment system.
    • Sending and receiving trackbacks.
  • File downloader
    • You can post, edit, delete, search and add tags.
    • You can stock any kinds of files which SQLite database can handle.
      All files will be stocked in SQLite database into separated tables to avoid overhead when downloading.
    • Archive searching with calendar.
    • MD5 hash generation.
    • Download counter
      You can check this download counter when logged in.
  • RSS Feeding
    • RSS2.0(Really Simple Sindycation)
    • Supports Podcast and Videocast
  • Flexible plugin architecture.
  • You can manage and create a static page contents with "Loggix Expander".
  • CSS style switcher

Requirements

Loggix runs on PHP5 or higher, OS independent, of course, runs on UNIX-like system such as Mac OS X, Linux, BSDs, and Windows.

Loggix is powered by SQLite. All tables are installed already so you don't need any "INSTALL" procedures. Easy!

FAQ

Q : Does Loggix run on PHP4.x?
A : No.
Why? : Loggix uses many functions added on PHP5 and use SQLite as a database engine.
Q : Do you have any plan to release MySQL driven Loggix?
A : No and never.
Why? : Loggix is built for
  1. Using Lightweight database engine
  2. Easy-to-install
  3. Easy-to-backup
and SQLite is the answer. It also using low-level detabase drivers (No MDB or any DB layers) for quick response. Feel free to create an other DB version of Loggix, if you want to. It's free software under New BSD Lisence!
